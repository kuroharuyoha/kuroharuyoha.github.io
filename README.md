# 深渊小钓 - 挂机钓鱼游戏

一个轻松有趣的网页挂机钓鱼游戏，采用自动化钓鱼机制，玩家可以升级能力来获得更好的收益。

## 游戏特性

### 核心玩法
- **自动钓鱼流程**：鱼钩自动甩下、等待鱼上钩、自动收杆的完整流程
- **进度系统**：背包容量达到上限时自动暂停，清空后继续钓鱼
- **背包系统**：每条鱼占5格背包容量，支持一键出售获得金币
- **升级系统**：三项主要能力升级，成本随着等级指数增长

### 游戏机制

#### 钓鱼流程
1. **下沉阶段**：鱼钩以当前速度匀速下沉到设定的最大深度
2. **等待阶段**：在该深度等待1-5秒的随机时间
3. **产鱼阶段**：在该深度范围内产生一条随机的鱼（种类和尺寸都随机）
4. **收杆阶段**：鱼钩以当前速度匀速提升回到海平面
5. **获得**：鱼进入背包，获得对应的金币（如果背包满了则暂停）

#### 背包系统
- **初始容量**：20格（可容纳4条鱼）
- **占用方式**：每条鱼无论种类或大小都占5格
- **一键出售**：将背包内所有鱼出售，获得总金币
- **暂停机制**：背包满后暂停自动钓鱼，直到清空背包

#### 升级系统
三项可升级的能力，每项都有20个等级：

| 能力 | 初始值 | 增量 | 基础花费 | 花费倍增 |
|------|--------|------|---------|---------|
| 最大深度 | 100m | 50m/级 | 50金币 | 1.5倍/级 |
| 移动速度 | 50m/s | 5m/s/级 | 50金币 | 1.5倍/级 |
| 背包容量 | 20格 | 10格/级 | 50金币 | 1.5倍/级 |

### 鱼类系统

#### 鱼的种类与分布

| 鱼种 | 活跃深度 | 大小范围 | 基础价格系数 | 颜色 |
|------|---------|---------|------------|------|
| 小鱼 | 0-50m | 5-15cm | 2金币/cm | 金色 |
| 金鱼 | 30-100m | 8-20cm | 3金币/cm | 红色 |
| 鲶鱼 | 60-150m | 15-30cm | 5金币/cm | 棕色 |
| 鲑鱼 | 100-200m | 20-35cm | 7金币/cm | 朱红色 |
| 深海鱼 | 150-300m | 25-40cm | 10金币/cm | 靛蓝色 |
| 幽灵鱼 | 250-400m | 30-50cm | 15金币/cm | 白色 |
| 恶魔鱼 | 300-500m | 35-60cm | 20金币/cm | 深粉红 |

#### 价格计算
- **线性关系**：同种鱼的价格与大小为线性关系
- **公式**：`获得金币 = 基础价格系数 × 鱼的大小`
- **例子**：10cm的金鱼 = 3 × 10 = 30金币

## 控制说明

### 游戏控制按钮
- **开始游戏**：开始钓鱼，首次点击后变为"继续"
- **暂停**：暂停游戏进行
- **重置游戏**：重置所有进度（需要确认）

### 能力升级
- 每项能力都有"升级"按钮
- 当金币不足或达到最大等级时，按钮禁用
- 升级后立即生效

### 背包管理
- **背包显示**：实时显示背包使用情况（已用格子/总容量）
- **一键出售**：将背包所有鱼出售，获得对应金币

## 游戏策略建议

### 初期策略（深度300m以内）
- 前期集中升级**背包容量**，扩大一次收获的鱼数量
- 适度升级**移动速度**以提高效率
- 等背包容量达到可接受水平后再升级深度

### 中期策略（深度300-700m）
- 平衡升级**最大深度**和**移动速度**
- 优先升级深度以解锁高价值鱼种
- 定期出售以积累升级资金

### 后期策略（深度深于700m）
- 以升级**最大深度**为主，解锁最高价值的鱼种
- 升级**移动速度**以提高整体效率
- 根据需求调整背包容量

## 技术说明

### 使用技术
- **HTML5**：游戏结构和UI
- **CSS3**：响应式设计和动画效果
- **Vanilla JavaScript**：游戏逻辑，无外部依赖
- **SVG**：游戏画布和动画

### 浏览器兼容性
- 支持所有现代浏览器（Chrome、Firefox、Safari、Edge）
- 需要启用JavaScript

## 开始游戏

### 方法1：直接打开HTML文件
```
双击 index.html 文件即可在默认浏览器中打开
```

### 方法2：使用Python HTTP服务器（推荐）
```bash
cd 游戏文件夹
python -m http.server 8000
```
然后在浏览器中访问 `http://localhost:8000`

### 方法3：使用其他服务器
任何支持HTTP服务的服务器都可以，比如Node.js的http-server等
